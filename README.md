# Simple Full Stack Greeting App

This is my first full stack project built using:

- HTML5
- CSS3
- JavaScript
- Spring Boot
- Maven

## 🚀 Features

- User enters name
- Frontend sends request to backend
- Backend processes request
- Displays: Hello, <UserName>

## 🛠 Tech Stack

Frontend:
- HTML
- CSS
- JavaScript (Fetch API)

Backend:
- Spring Boot
- REST API
- Maven

## ▶ How to Run

1. Clone repository
2. Run:
   mvn spring-boot:run
3. Open browser:
   http://localhost:8080

This project was built as practice while learning full stack development.
